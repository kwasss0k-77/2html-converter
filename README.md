# 2html-converter
Converts simpler code (see README.txt) to html. Instrument, to write html code if you dont know it. You write a command from command list in README.txt and its converting to html code. This code like new program language XD


## Instructions
### download
Just click releases and download latest version.

### Usage
Open README.txt in main folder and find needed command. Following the rules, write it on new line.
